************
Requirements
************

uJet is compatible with Umbraco 7, target framework .NET 4.5 and requires Full Trust.

NuGet Requirements
------------------
The uJet NuGet targets Umbraco CMS 7.2.8 and above.

Development Requirements
------------------------
The uJet source code is written in C# 5.0 (.NET 4.5) using Visual Studio 2015.
